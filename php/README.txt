EBI Web Services PHP Clients
============================

A set of sample clients for the EBI Web Services written in PHP.

Directory     Description
---------------------------
nusoap/       Clients for SOAP services using nuSOAP
pear_soap/    Clients for SOAP services using PEAR SOAP
php_file/     Clients for REST services using PHP file_get_contents
php_soap/     Clients for SOAP services using PHP SOAP

See http://www.ebi.ac.uk/Tools/webservices/ for more details.
