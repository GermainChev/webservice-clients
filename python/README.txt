EBI Web Services Python Clients
===============================

A set of sample clients for the EBI Web Services written in Python.

Directory     Description
---------------------------
urllib/       Clients for REST services using urllib/urllib2
soappy/       Clients for SOAP services using SOAPpy
zsi/          Clients for SOAP services using ZSI

See http://www.ebi.ac.uk/Tools/webservices/ for more details.
