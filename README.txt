EBI Webservices Clients
=======================

Sample Clients
--------------

A collection of sample clients for the EBI webservices in a range of 
programming languages:

Language           Directory
--------           ---------
C#                 csharp/
Java               java/
Perl               perl/
PHP                php/
Python             python/
Ruby               ruby/
Visual Basic .NET  vb.net/

A selection of utility scripts are provided in the util/ directory.

Sample Workflows
----------------

A selection of sample workflow descriptions showing how to use the 
services can be found in the following directories:

Workflow Engine  Directory
---------------  ---------
Taverna          taverna/
